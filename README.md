# ValueDesigner

Landing page for [valuedesigner.io](https://valuedesigner.io)

Built with React + Vite.

---

## Setup

```bash
npm install
npm run dev
```

Site runs at `http://localhost:5173`

---

## Before you go live

1. **Booking URL** — open `src/App.jsx` and replace the `BOOKING_URL` constant at the top with your real Calendly (or cal.com) link.

2. **Deploy** — connect this repo to [Vercel](https://vercel.com) or [Netlify](https://netlify.com). Both auto-detect Vite. Zero config needed.
